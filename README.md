# niconico-usercss
Userstyles to improve UX of NicoNico

### Seiga Illusts into 5 Columns
Arranges the thumbnails of NicoNico Seiga search/user page in 5 columns.

ニコニコ静画の検索ページとユーザーページのサムネイルを 5 列表示に変更します。
